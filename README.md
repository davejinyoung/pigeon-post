# Pigeon Post
Pigeon Post is a web application designed to connect seamlessly with your Gmail inbox and provide clear, concise summaries of your emails using AI. Built with Angular on the frontend and Django on the backend, it utilizes the Gmail API and integrates with locally hosted large language models via Ollama for secure, offline summarization.
